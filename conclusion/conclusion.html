Evaluation

This project shows a working wildfire risk monitoring system that integrates sensors, a microcontroller, a server, and a machine learning model. The system collects temperature and humidity data using the Raspberry Pi Pico, transmits this data to a FastAPI server, and generates a wildfire risk prediction. Through developing this project, I gained experience in embedded programming, server development, and integrating machine learning into a real-time system.

However, the project also has several limitations. The machine learning model was trained using a Californian wildfire dataset, which does not perfectly represent environmental conditions in other regions. As a result, the predictions may not be fully accurate outside of similar climates. Additionally, the model only considers temperature and humidity as input features, while real wildfire risk is influenced by many other factors such as wind speed, vegetation type, and soil moisture. However this could easily be fixed by adding more sensors.

Another limitation is the emergency alert button, which sends a signal over Wi-Fi. In real deep forest environments, Wi-Fi connectivity may not be available. I could further improve it by deploying a network of microcontrollers that communicate with each other. (My initial idea. But I only had access to one microcontroller so I couldn't test it).

Building the website was the most enjoyable part. It felt refreshing to step away from modern frameworks and libraries like react, bootstrap (Due to the required file structure in the brief) and instead build everything from scratch. This approach reminded me of when I first started coding and helped put things into perspective at its core, programming is simply about building things. I also got to learn a lot of new things particularly using ML models and how to incorporate them in a proper system.

Overall, I believe the project achieved its main objective while also providing me valuable insight into the challenges of building and deploying a real-world system.
